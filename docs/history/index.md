# Triplet Phase Lock

A minimal pipeline:

```
structure → variation → selection
```

Stages:

- **Expand (Π)** → global structure persists  
- **Extend (π)** → local drift emerges  
- **Resist (Π)** → strict thresholds select stability  

---

## Core result

```
directional drift ↑ ⇒ strict resistance ↓
```

This repository demonstrates that:

- structure remains invariant under perturbation  
- drift appears at the local (step-to-step) level  
- thresholding converts drift into survival or failure
