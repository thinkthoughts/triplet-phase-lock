# Triplet Map

Triplet phase-lock organizes construction into staged triplets:

\[
\Pi^{(n)} = (Pi_i, Pi_{i+1}, Pi_{i+2})
\]

---

## Stages

### Π⁽⁰⁾ — expand (0Pi–2Pi)

- IA: Z0 → Z1 → Z2  
- VC: A → B  

- unconstrained → collapse → assignment  
- anchor → bilateral  

---

### Π⁽¹⁾ — extend (3Pi–5Pi)

- IA: Z2 → Z3  
- VC: C → D  

- assignment → denial  
- constraint → domain consistency  

---

### Π⁽²⁾ — resist (6Pi–8Pi)

- IA: Z3 → Z4  
- VC: C → D  

- denial → mixed inconsistency  
- constraint → stable comparison  

---

### Π⁽³⁾ — synthesis (9Pi–11Pi)

- IA: Z4  
- VC: A + B + C + D  

- mixed inconsistency → unstable synthesis  
- full construction → stable synthesis  

---

## Summary

- IA accumulates inconsistency  
- VC accumulates structure  

\[
ZOS \Rightarrow IA \qquad
GOS \Rightarrow VC
\]
